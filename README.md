# HR Attrition Analytics Dashboard

## Overview

This project presents an interactive HR Attrition Analytics Dashboard developed in Microsoft Excel to analyze employee turnover and identify key factors contributing to attrition.

## Features

* Dynamic KPI Cards
* Interactive Slicers
* Pivot Tables
* GETPIVOTDATA Functions
* Interactive Charts
* Business Insights
* Executive Dashboard Layout

## KPIs

* Total Employees
* Total Attrition
* Attrition Rate

## Analysis Performed

* Attrition Rate by Department
* Attrition Rate by Job Role
* Attrition Rate by Age Group
* Attrition Rate by Gender
* Attrition Rate by Education Field
* Attrition Rate by Overtime
* Attrition Rate by Years in Current Role

## Key Insights

* Sales Representatives have the highest attrition rate.
* Employees working overtime have significantly higher attrition than employees who do not work overtime.
* Employees aged 18–27 exhibit the highest attrition.
* The Sales department records the highest employee attrition.
